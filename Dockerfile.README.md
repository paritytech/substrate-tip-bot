# substrate-tip-bot

## A GitHub App built with Probot that can submit tips on behalf of a Substrate based network.

To run the bot via Docker:

```
$ docker run \
    -e APP_ID=<app-id> \
    -e PRIVATE_KEY=<pem-value> \
    paritytech/substrate-tip-bot
```
